# JediMindAgent

Проект в разработке.

JediMindAgent - ИИ-ассистент для диагностики текущего состояния пользователя и подбора практик из книги «Путь джедая» с учетом контекста диалога.

## Что планируется
- чат с агентом для диагностики состояния и уровня усталости
- RAG-поиск по материалам книги и упражнениям
- адаптация упражнений под контекст пользователя
- сохранение истории сообщений и прогресса по упражнениям
- REST API и простой веб-интерфейс для тестирования
